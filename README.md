# Scraping


run it in the following way
```
python scraping.py --vol_start=383 --vol_end=383 -o output_filepath
```
vol_start: the oldest volume to be scraped. The oldest avaliable journals in 1997 corresponds with 336.

vol_end: the latest volume to be scraped. The latest avaliable journals in 2020 corresponds with 383.

o: output file path. Please use directory, instead of the file name.
